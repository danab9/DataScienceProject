This folder contains all GO/pathway enrichments

subfolder: 
- "data" contains the list of up/downregulated genes 
- "DAVID_results" contains all outcomes from the online tool DAVID 
- "plots" contains all plots that have been created by the script Enrichment_Visualization.Rmd as well as  heatmaps for the GO "regulation of transcription"  (RT_heatmap.png, RT_heatmapNew.png)

R scripts: 

- Enrichment_Visualization.Rmd shows how the GO and pathway enrichments were done as well as visualizing 
- GODavid_differentialexpression.R shows visualization of GO term (David)


furthermore, we checked the results by reading AD literatures, the summary is written down in enrichment_linked_to_papers.odt
